### 1.git add readme.txt （可以多次操作）
### 2.git commit -m "wrote a readme file111" （上一步的多次操作，这里做一次即可全部commit）
### 3.1-git push -u origin master (第一次向远程推本地代码) 
> 加上了-u，Git不但会把本地的master分支内容推送的远程新的master分支，还会把本地的master分支和远程的master分支关联起来，在以后的推送或者拉取时就可以简化命令。
